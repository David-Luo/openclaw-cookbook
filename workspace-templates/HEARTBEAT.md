# HEARTBEAT.md — 定时任务与自主行为配置

## 说明

HEARTBEAT.md 配置 Agent 的自主行为节律。
OpenClaw 会根据此文件的 cron 规则，在指定时间自动唤醒 Agent 执行任务。

## 每日晨报

```yaml
cron: "0 9 * * 1-5"  # 工作日早上9点
```

### 任务内容
- 检查 Git 仓库最近 24 小时的提交和 PR
- 汇总待处理的 Issue 和 Review 请求
- 检查 CI/CD 流水线状态
- 生成简要日报推送

## 周度架构健康检查

```yaml
cron: "0 10 * * 1"  # 每周一上午10点
```

### 任务内容
- 扫描项目依赖更新和安全漏洞
- 检查 TODO/FIXME/HACK 标记的数量变化趋势
- 统计本周技术债变化
- 生成周度架构健康报告

## 文档同步检查

```yaml
cron: "0 17 * * 5"  # 每周五下午5点
```

### 任务内容
- 检查 API 文档与实际接口的一致性
- 检查 README 和架构文档是否需要更新
- 标记过期文档

## 注意事项

- 所有自动任务仅生成报告和建议，不自动修改代码或配置
- 如需调整执行频率，修改对应的 cron 表达式
- 任务执行结果会通过配置的通知渠道推送
